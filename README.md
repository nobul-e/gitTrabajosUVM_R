# gitTrabajosUVM

Este es un proyecto para las prácticas de la UVM. Aquí aprenderemos Git y GitHub.

## Objetivos:
- Aprender el uso básico de Git
- Configurar un repositorio en GitHub
- Manejar control de versiones

## Funcionalidades:
- Inicialización de repositorio
- Configuración de README
- Conexión a GitHub

## Nueva funcionalidad
Esta funcionalidad agrega ejemplos de Git en la rama feature.
